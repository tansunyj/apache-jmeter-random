另：随机数都是在ThreadLocal的

#### 使用说明

如：
a. 极小值1，极大值10，占比20；
b. 极小值40，极大值70，占比30；
c. 极小值100，极大值200，占比50；
数据格式：%.2f

最终会生成：
1.范围在1~10之间的浮点数，占总数约20%
2.范围在40~70之间的浮点数，占总数约30%
3.范围在100~200之间的浮点数，占总数约50%
